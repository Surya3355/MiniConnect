# Product Requirements Document

## Project

MiniConnect

## Vision

MiniConnect is a peer-to-peer communication engine that enables Android devices to communicate directly using Wi-Fi Direct without requiring Internet access.

## Problem Statement

Many communication applications depend on cloud connectivity.

MiniConnect aims to provide reliable offline communication between nearby Android devices.

## MVP Goals

- Discover nearby devices
- Establish peer connection
- Exchange text messages
- Handle disconnects gracefully

## Out of Scope

- User accounts
- Cloud backend
- Voice calls
- File transfer
- Push notifications
