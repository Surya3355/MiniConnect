Milestone 1 - Wi-Fi Direct Infrastructure

# Objective

Create the initial networking layer responsible for interacting with Android's Wi-Fi Direct framework.

This milestone only initializes the Android Wi-Fi Direct components.

No peer discovery, connection, or socket communication should be implemented.

---

# Background

MiniConnect follows a layered architecture.

The UI must never communicate directly with Android's WifiP2pManager.

All Android Wi-Fi Direct APIs must be encapsulated inside a dedicated manager.

Future milestones will build on this manager.

---

# Architecture

```
Compose UI
      │
      ▼
ConnectionViewModel
      │
      ▼
WifiDirectManager
      │
      ▼
WifiP2pManager
      │
Android Wi-Fi Direct Framework
```

---

# Scope

Included

- Initialize WifiP2pManager
- Initialize Channel
- Validate initialization
- Provide a clean API for future milestones

Not Included

- Peer discovery
- Device connection
- BroadcastReceiver
- TCP sockets
- Permissions
- Compose UI
- ViewModel

---

# Files

Create

```
WifiDirectManager.kt
```

Do not create any other production classes.

---

# Public API

The class must expose exactly these public methods.

```kotlin
class WifiDirectManager {

    fun initialize(context: Context): Result<Unit>

    fun isInitialized(): Boolean

}
```

No additional public methods.

Future milestones will introduce:

- discoverPeers()
- connect()
- disconnect()

Those methods must NOT exist yet.

---

# Internal State

The class may internally maintain:

```kotlin
private var wifiP2pManager: WifiP2pManager?

private var channel: WifiP2pManager.Channel?

private var initialized: Boolean
```

These fields must remain private.

---

# Responsibilities

WifiDirectManager is responsible for

- obtaining WifiP2pManager
- creating the Channel
- validating initialization

WifiDirectManager is NOT responsible for

- UI
- Toasts
- Logging to screen
- Compose
- Activity lifecycle
- Socket communication

---

# Error Handling

If WifiP2pManager cannot be obtained

Return

```
Result.failure(...)
```

Do not crash.

Do not swallow exceptions.

---

# Acceptance Criteria

- Project builds successfully.
