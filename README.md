# MiniConnect

MiniConnect is a modern Android peer-to-peer communication engine built using Wi-Fi Direct and TCP sockets.

The project is being developed using production-quality engineering practices, including architecture documentation, sprint planning, Git workflows, and clean code principles.

---

## Vision

Build a reliable offline communication engine that can power applications such as RiderConnect.

---

## Technology Stack

- Kotlin
- Jetpack Compose
- MVVM Architecture
- Wi-Fi Direct
- TCP Sockets
- Git & GitHub
- Android Studio
